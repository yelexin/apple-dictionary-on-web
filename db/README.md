# Extract the Dictionaries.zip File in the Folder

[Download](https://drive.google.com/drive/folders/1LP837oyXd22Kgc1S_l6VEnc69qwkuL1O?usp=drive_link) the dictionary database zips and unzip them.

Put `ChineseEnglishDictionary.db` and `NewOxfordAmericanDictionary.db` and etc. **directly** in this folder.

They are sqlite3 database with indices.
